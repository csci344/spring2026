Topics > 6. JavaScript: Interacting with the DOM (Document Object Model)

In this unit, we will use JavaScript to manipulate the Document Object Model (DOM) and respond to user-initiated events, and dynamically generate content from data. By the end of this section, you will know enough to start building simple, interactive web applications.

Study Questions / Concepts

  1. Remember that DOM objects and methods are only used in "client-side" programming (within a browser). They are not supported in Node.js
  2. Be able to use JavaScript to manipulate HTML and CSS dynamically (usually through event handlers).
  3. Practice targeting and changing DOM elements using the DOM manipulation worksheet
  4. Please refer to the working with the DOM section of the JavaScript cheatsheet and make sure you know how to interact with the DOM.
